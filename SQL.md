# Summary of Relational Databases and SQL

Relational databases are a type of database that organizes data into one or more tables with a unique key identifying each row. SQL (Structured Query Language) is a language used to communicate with relational databases and manipulate data.

In order to work with relational databases and SQL, one should be familiar with the following concepts and commands:

-   SELECT statement: used to retrieve data from one or more tables
-   WHERE clause: used to filter data based on certain criteria
-   JOIN: used to combine rows from two or more tables based on a related column
-   GROUP BY: used to group rows based on one or more columns
-   ORDER BY: used to sort rows based on one or more columns
-   INSERT INTO: used to insert new rows into a table
-   UPDATE: used to update existing rows in a table
-   DELETE: used to delete rows from a table
-   CREATE TABLE: used to create a new table with specified columns and data types
-   ALTER TABLE: used to modify an existing table, such as adding or deleting columns
-   DROP TABLE: used to delete an entire table
