# Bash Command Line Tutorials Summary

## The Command Line - What is it, how does it work and how do I get to one:

-   The command line is a text-based interface used to interact with the operating system and execute commands.
-   To access the command line, you can open the Terminal application on macOS, or use the Command Prompt or PowerShell on Windows.
-   The terminal provides a way to navigate directories, manage files, and run various programs using command-line arguments and options.

## Basic Navigation - An introduction to the Linux directory system and how to get around it:

-   The Linux directory system is structured as a hierarchy, with the root directory at the top and all other directories branching off from it.
-   The `cd` command is used to change directories, and the `ls` command is used to list the contents of a directory.
-   The `..` symbol represents the parent directory, and the `.` symbol represents the current directory.

## More About Files - Find out some interesting characteristics of files and directories in a Linux environment:

-   Files in Linux have permissions that determine who can read, write, or execute them.
-   The `chmod` command is used to change file permissions.
-   The `file` command can be used to determine the type of a file.

## Manual Pages - Learn how to make the most of the Linux commands you are learning:

-   The `man` command is used to access manual pages, which provide documentation for various commands.
-   Manual pages include a description of the command, its syntax, available options, and examples of its usage.

## File Manipulation - How to make, remove, rename, copy and move files and directories:

-   The `touch` command is used to create new files, and the `mkdir` command is used to create new directories.
-   The `rm` command is used to remove files and directories.
-   The `mv` command is used to rename files and directories, as well as move them to a different directory.
-   The `cp` command is used to copy files and directories.
